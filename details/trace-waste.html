Trace Waste

A digital compliance platform that brings traceability and accountability to Nigeria's e-waste value chain.
Live
Problem

Nigeria's e-waste sector operated in the dark open cable burning, unsafe material recovery, and untraceable disposal were commonplace. Regulators had no real-time data, businesses had no way to demonstrate compliance, and informal collectors had no structured path into legitimate systems which needed a digital solution that matched the complexity of the problem.

Solution

Built the platform for W3 Eco-Friendly Ltd, which Laravel powers the compliance engine chain-of-custody records, audit scheduling, compliance scoring, and ESG report generation. React handles the frontend dashboards for three distinct user types: businesses requesting compliance, recyclers logging pickups, and regulators accessing real-time audit data.
